What to Do Next

Step 7 of 7

You've made it through the guide. You now know more about using Claude effectively than most people who use it every day. Here's where to go from here.

Keep practising

The single best thing you can do is use Claude regularly for real tasks in your life. Not test questions — actual things you need. Writing, planning, learning, solving problems. The more you use it, the more natural it becomes.

Some ideas to get started:

  • Next time you need to write an email, ask Claude to help draft it
  • When you're confused about something, ask Claude to explain it
  • When you have a decision to make, use Claude as a thinking partner
  • When you have a long document to read, paste it in and ask for a summary

Claude chat vs Claude Code

You've been using Claude chat — the version you access at claude.ai. There's also something called Claude Code, which is a different tool.

Claude chat (claude.ai) — What you've been using. A conversation interface. Best for writing, thinking, learning, answering questions. Anyone can use it.
Claude Code — A tool for software developers. It runs in a terminal (a text-based window on your computer) and helps you write and edit code. This is what was used to build this website. You don't need it unless you want to build software.

If you're curious about Claude Code and want to try it, the best way to start is by building something small — like a website. Which is exactly what this site was made with.

Things worth exploring

  • Projects — Claude.ai lets you create "Projects" to keep related conversations together. Useful for ongoing work.
  • Uploading files — You can upload PDFs, Word docs, and images directly to Claude and ask questions about them.
  • Custom instructions — You can tell Claude things about yourself once (like your job, preferences, or writing style) so it keeps that in mind every time.

You're ready

There's no more guide to read. The rest is just doing. Go ask Claude something real today — and remember, there's no wrong way to use it.
