# Website Upgrade Design — Learn Claude

**Date:** 2026-03-10

## Purpose

Transform the existing 4-page starter site into a complete beginner guide for people new to Claude. Clear journey, practical examples, warm tone.

## Approach

Improve all 4 existing pages and add 4 new pages. One shared CSS file gets polished. Plain HTML + CSS, no new tools.

## Page Structure (in journey order)

| # | Page | File | Status |
|---|------|------|--------|
| 1 | Home — Start Here | index.html | Improve |
| 2 | What is Claude? | what-is-claude.html | Improve |
| 3 | Getting Started | getting-started.html | Improve |
| 4 | How to Ask Better Questions | better-questions.html | New |
| 5 | Common Beginner Mistakes | mistakes.html | New |
| 6 | Everyday Prompt Patterns | prompts.html | New |
| 7 | Follow-up & Refining Answers | follow-up.html | New |
| 8 | What to Do Next | next-steps.html | New |

Every page ends with a "Next step →" link to the next page in the journey.

## Content

### Improved pages
- **Home** — Redesigned as "Start Here". Numbered journey path showing all 8 pages. Warm tone.
- **What is Claude?** — Everyday examples, Claude vs Google comparison, best first things to try.
- **Getting Started** — Best first prompts with bad/better/best examples. Starting over vs continuing tip.

### New pages
- **Better Questions** — Tone, context, format in plain English. Bad/better/best comparisons.
- **Mistakes** — 6–8 common beginner mistakes with explanations.
- **Prompt Patterns** — Real templates for writing, summarising, brainstorming, planning, learning.
- **Follow-up** — How to refine answers. When to start a new conversation.
- **Next Steps** — Claude Code vs Claude chat. What to explore next.

## Visual Design

- Improved spacing and section dividers
- Cards with subtle left-border accent for tips/examples
- Bad/better/best labels use red/yellow/green colours
- Nav updated with all 8 pages
- "Next step →" button at the bottom of every page
